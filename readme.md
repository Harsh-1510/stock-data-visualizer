# Stock Market Data Visualization

This web application displays stock market data from a CSV file. It shows a list of companies on the left side and displays a chart with the selected company's data on the right side.

## Features

- Display a list of companies
- Interactive charts showing Open, Close, High, and Low values
- Detailed company information including trading volume, PE ratio, etc.
- Responsive design using Bootstrap

## Tech Stack

- **Frontend**: HTML, CSS, Bootstrap, Chart.js
- **Backend**: Node.js with Express

## Setup Instructions

1. Clone the repository:
    git clone https://github.com/your-username/stock-market-app.git
    cd stock-market-app
2. Install dependencies:
    npm install
3. Place your CSV file in the data folder as dump.csv
4. Start the server:
    node server.js
5. Open your browser and navigate to http://localhost:3000
